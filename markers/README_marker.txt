PETROPAVLOVSKAYA_DEMO
======================

Локальный демо-проект WebAR (A-Frame + AR.js) для сцены "Петропавловская крепость — Основание города".

Кратко:
- Откройте index.html на мобильном устройстве (браузер должен поддерживать getUserMedia).
- Разрешите доступ к камере.
- Наведите камеру на маркер Hiro (инструкция ниже).
- Нажмите "Слушать Петра I" или "Поднять флаг".

Marker (маркер Hiro):
- В демо используется публичный файл patt.hiro, расположенный на raw.githack.
- Для офлайн-использования скачайте файл patt.hiro из репозитория AR.js:
  https://github.com/AR-js-org/AR.js/blob/master/aframe/examples/marker-training/examples/pattern-files/patt.hiro
  Сохраните его в папку markers/ как markers/patt.hiro и измените URL в index.html соответственно:
    <a-marker type="pattern" url="markers/patt.hiro">
- Также можно использовать любую распечатанную версию Hiro marker (поиск "hiro marker" выдаст изображение).

Запуск локально:
1) Запустите простой HTTP-сервер в папке проекта. Например:
   python3 -m http.server 8000
   или
   npx http-server

2) Откройте на мобильном устройстве: http://<ваш-компьютер-IP>:8000

Примечания:
- В демо используются простые A-Frame примитивы вместо 3D моделей (.glb) для простоты развёртывания.
- Файлы звуков сгенерированы автоматически (короткие WAV с простыми тонами).
- Для реального проекта замените примитивы на оптимизированные glTF/glb модели и добавьте локальные или CDN-ресурсы.

Enjoy!
